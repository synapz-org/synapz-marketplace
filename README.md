# Synapz Claude Code Marketplace

A curated collection of Claude Code skills for Bittensor ecosystem tools, social media automation, and AI infrastructure management.

## Installation

Add the marketplace to Claude Code:

```
/plugin marketplace add synapz-org/synapz-marketplace
```

Then install any skill:

```
/plugin install typefully@synapz
/plugin install btcli@synapz
/plugin install basilica-miner@synapz
```

## Available Skills

### Social Media & Automation

| Skill | Description |
|-------|-------------|
| **typefully** | Manage social media through Typefully API v2 - drafts, scheduling, cross-posting across X/LinkedIn/Threads/Bluesky/Mastodon |

### Bittensor CLI & Infrastructure

| Skill | Description |
|-------|-------------|
| **btcli** | Bittensor CLI - wallet management, subnet registration, staking, and network operations |
| **basilica-cli** | Rent GPUs for ML training on Basilica decentralized compute |
| **hippius** | Hippius decentralized storage on Bittensor Subnet 75 - IPFS/S3 operations |

### Bittensor Miner Skills
